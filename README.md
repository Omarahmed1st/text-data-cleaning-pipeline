# Text Data Cleaning Pipeline for NLP

This project demonstrates a simple, reusable pipeline for cleaning messy text data such as student essays and free-text responses.

## Features
- Remove special characters and noise
- Normalize text (lowercase, spacing)
- Remove duplicates
- Basic filtering of low-quality text

## Use Case
Designed for preparing datasets for NLP models (classification, fine-tuning, etc.)

## How to Run
```bash
pip install -r requirements.txt
python pipeline.py
